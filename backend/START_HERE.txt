═══════════════════════════════════════════════════════════════════════════
PYTEST BASICS AT A GLANCE
═══════════════════════════════════════════════════════════════════════════

WHAT IS PYTEST?
    Testing framework for Python
    Makes writing tests easy and fun
    Automatically discovers and runs tests

BASIC STRUCTURE:
    def test_something(fixtures_here):
        # ARRANGE: Setup data
        # ACT: Do something
        # ASSERT: Check result
        assert result == expected

FIXTURE (Reusable Setup):
    @pytest.fixture
    def test_user():
        return User(email="test@example.com")
    
    def test_something(test_user):
        # test_user automatically available

ASSERTION (Check Results):
    assert response.status_code == 201
    assert "email" in response.json()
    assert "password" not in response.json()

MARK (Categorize Tests):
    @pytest.mark.unit
    def test_password_hashing():
        # Fast, no database
    
    @pytest.mark.integration
    def test_login_endpoint():
        # Uses HTTP and database

RUN TESTS:
    pytest                                  # All tests
    pytest -v                              # Verbose (show names)
    pytest -m unit                         # Only unit tests
    pytest --cov=app --cov-report=html     # With coverage
